# 3296. Minimum Number of Seconds to Make Mountain Height Zero

## Problem Statement

You are given:

- An integer `mountainHeight` representing the height of a mountain.
- An array `workerTimes` where `workerTimes[i]` is the base time for worker `i`.

Workers work **simultaneously**.

If worker `i` reduces the mountain by `x` units of height, the time taken is:


workerTimes[i] * 1 +
workerTimes[i] * 2 +
workerTimes[i] * 3 +
...
workerTimes[i] * x


Return the **minimum number of seconds** required so that the total reduction by all workers reduces the mountain height to **0**.

---

# Core Mathematical Observation

The time for a worker removing `x` height is a series:


w + 2w + 3w + ... + xw


Factor out `w`:


w(1 + 2 + 3 + ... + x)


The sum of the first `x` natural numbers:


1 + 2 + 3 + ... + x = x(x + 1) / 2


So the time becomes:


time = w * x(x + 1) / 2


---

# Key Insight

Workers operate **simultaneously**, so the total time required is determined by:


max(worker finishing time)


We want to **minimize this maximum time**.

This becomes a **scheduling / feasibility problem**.

---

# Pattern Recognition

When a problem asks:

- **Minimum time**
- Workers operate **simultaneously**
- Tasks can be distributed

You should ask:


If we give workers T seconds,
can they finish the job?


This converts the problem into a **YES / NO feasibility check**.

If `T` works, then `T+1` will also work.

So the feasibility looks like:


F F F F T T T T


This monotonic behavior suggests:


Binary Search on Answer


---

# Binary Search Strategy

We search for the **smallest time `T`** such that workers can remove the mountain.

Search range:


low = 1
high = slowestWorker * H(H + 1) / 2


Worst case → one slow worker removes everything.

---

# Feasibility Check

For each worker we compute:


max height x such that

w * x(x + 1) / 2 <= T


Rearranging:


x(x + 1) <= 2T / w


Solving the quadratic inequality:


x = floor((-1 + sqrt(1 + 8T/w)) / 2)


Add heights from all workers.

If:


sum >= mountainHeight


then the time `T` is feasible.

---

# Java Implementation

```java
class Solution {

    public boolean canFinish(long time, int mountainHeight, int[] workerTimes) {
        long removed = 0;

        for (int w : workerTimes) {

            long val = (long) Math.sqrt(1 + (8L * time) / w);
            long x = (val - 1) / 2;

            removed += x;

            if (removed >= mountainHeight) {
                return true;
            }
        }

        return false;
    }

    public long minNumberOfSeconds(int mountainHeight, int[] workerTimes) {

        long low = 1;
        long maxWorker = 0;

        for (int w : workerTimes) {
            maxWorker = Math.max(maxWorker, w);
        }

        long high = maxWorker * (long) mountainHeight * (mountainHeight + 1) / 2;

        while (low < high) {

            long mid = low + (high - low) / 2;

            if (canFinish(mid, mountainHeight, workerTimes)) {
                high = mid;
            } else {
                low = mid + 1;
            }
        }

        return low;
    }
}

```
